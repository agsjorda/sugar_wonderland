# Base Template

Steps to create a new game
- Copy
- Rename "name" in package.json same with the root folder name
- ```npm install```
- ```npm run dev```

# Branch naming convention
- Only dev, staging, and main are for CD/CI
- Branch prefix should be the task ticket code from JIRA, ex: GST-1
- Squash and Merge, commit message should be the task ticket
